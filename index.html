Stut Application

L'utilitaire qui facilite la gestion des données personnelles pour améliorer la communication entre les profs et leurs élèves.

Réalisé pour les enseignants par un élève de BTS ERPC

J'ai toujours trouvé la communication élèves / professeurs mal optimisé. Pour palier à celà j'ai développé un outil gratuit qui permet de stocker et gérer les informations concernant les élèves
